# React Note-Taking App in Trae (New Solomon)
### Description
This is a simple note-taking application built using React. It allows users to create, edit, and delete notes.

- all notes are currently stored in local storage

##### Tech Stack
- BlockNote
- Shadcn UI
- DockView
- Godot Engine (Canvas)
